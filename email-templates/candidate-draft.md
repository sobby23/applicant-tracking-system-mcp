# Candidate outreach email template

The **actual template** used for draft emails is in **`candidate-draft.txt`** in this folder. Edit that file to change wording.

**Placeholders** (filled by the app):

- `{candidate_name}` — Applicant's first name.
- `{job_role}` — Human-readable job title (e.g. "Senior Software Engineer"), taken from the first heading of the job's `.md` file.
- `{body}` — Middle content generated by the evaluator (2–4 sentences: next step or polite decline).
- `{recruiter_name}` — Set in `.env` as `RECRUITER_NAME`. Falls back to "The Hiring Team" if not set.

Keep the placeholders exactly as above in `candidate-draft.txt`.
